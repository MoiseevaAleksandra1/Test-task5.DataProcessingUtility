# Тестовое задание 5. Создание программы «Утилита обработки данных»

В качестве входных данных принимается список клиентов оператора связи. Данные о клиентах следующие: 
- **ID** - идентификатор клиента
- **ФИО** - фамилия, имя, отчество
- **Дата рождения** - в формате ГГГГ.ММ.ДД
- **Номер телефона** - контактный номер
- **Тариф** - используемый тарифный план
- **Кол-во звонков в месяц** - количество минут разговоров
- **Кол-во СМС в месяц** - количество отправленных сообщений

## Проверка данных
Данные проверяются и разделяются на **корректные** и **некорректные**. Проверка осуществляется по следующим признакам:

### ФИО
- Проверка полноты (3 части: Фамилия Имя Отчество)
- Заглавные первые буквы
- Допустимые символы (кириллица и дефис)

### ID
- Уникальность
- Непустое значение
- Положительное число

### Дата рождения
- Корректность формата (ГГГГ.ММ.ДД)
- Не будущая дата
- Возраст более 13 лет
- Возраст менее 140 лет

### Номер телефона
- Форматы (+7XXXXXXXXXX или 8XXXXXXXXXX)
- Только допустимые символы

### Тариф
- Проверка на допустимые значения:
  - Безлимит
  - Эконом
  - Бизнес
  - Стандарт
  - Премиум

### Количество звонков/SMS
- Неотрицательные значения
- Реалистичные пределы

## Агрегированная статистика
Просчитывается следующая статистика:

### Анализ возраста
- Минимальный возраст
- Максимальный возраст
- Средний возраст
- Распределение по возрастным группам

### Анализ операторов
- Частотность кодов операторов связи

### Анализ тарифов
- Распределение тарифных планов
- Взаимосвязь с возрастными группами

### Расчет средних значений
- Среднее количество звонков по тарифам
- Среднее количество SMS по тарифам

## Отчет
Создается отчет со следующей информацией:
- Общее количество записей
- Количество корректных и некорректных записей
- Список ошибок с указанием причины

Полный список записей, а также вся полученная информация записывается в выходной файл `Report.txt`.
